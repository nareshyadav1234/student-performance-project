# Student Performance Analysis and Prediction

This project analyzes a dataset of 1,000 students to understand the factors affecting academic success and predicts final grades using Machine Learning.

## 🚀 Key Features
- **Data Cleaning:** Handled missing values using forward fill and removed duplicates.
- **Exploratory Data Analysis (EDA):** Visualized the impact of study hours, attendance, and parental support on grades.
- **Predictive Modeling:** Implemented a Linear Regression model.

## 🛠️ Tech Stack
- **Languages:** Python
- **Libraries:** Pandas, NumPy, Matplotlib, Seaborn, Scikit-learn
- **Tools:** VS Code, Jupyter Notebook

## 📊 Project Insights
- **Average Grade:** 80.0
- **Average Study Hours:** 2.42 hours/day
- **Model Performance:** The Linear Regression model yielded an R² Score of approximately **-0.005**. 
  - *Observation:* The low score indicates that the relationship between features and grades is non-linear, suggesting that more complex models (like Random Forest) could be used for better accuracy in future versions.
